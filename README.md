# Ansible Role: OpenVZWebPanelDaemon

An Ansible role that install [OpenVZ Web Panel](http://code.google.com/p/ovz-web-panel/) Daemon on openvz hosts

## Example Playbook
```sh
---
- hosts: openvzhosts
  gather_facts: no     
  roles:
     - OpenVZWebPanelDaemon
```
