# Ansible Role: OpenVZWebPanelDaemon

An Ansible role that install OpenVZ WebPanel Daemon on openvz hosts
[OpenVZ Web Panel](http://code.google.com/p/ovz-web-panel/) 

## Example Playbook
```sh
---
- hosts: openvzhosts
  gather_facts: no     
  roles:
     - OpenVZWebPanelDaemon
```
